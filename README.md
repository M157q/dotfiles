#rcfiles
Just bunch of my rcfiles

## Usage
Make sure have installed git.

### Install
1. `git clone https://github.com/M157q/rcfiles.git ~`
2. `cd ~/rcfiles`
3. `make install`

### Update
1. `cd ~/rcfiles`
2. `git pull`
3. `make clean`
4. `make install`

### Uninstall
1. `cd ~/rcfiles`
2. `make clean`
3. `rm -rf ~/rcfiles`
