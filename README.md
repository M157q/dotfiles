# rcfiles
Just a bunch of my dotfiles.

## Usage
Make sure you have git installed and the network connection is fine.

### Install
1. `git clone https://github.com/M157q/rcfiles.git ~`
2. `cd ~/rcfiles`
3. `make install`

### Update
1. `cd ~/rcfiles`
2. `git pull`
3. `make update`

### Uninstall
1. `cd ~/rcfiles`
2. `make clean`
3. `rm -rf ~/rcfiles`
